**Recursos Educacionais Abertos - UTFPR-CP 2018**

**Projeto:** Boas práticas de Engenharia de Software em disciplinas introdutórias de programação utilizando a plataforma Arduino

**Coordenador:** André Takeshi Endo

**Aluno:** Guilherme Gonçalves Borges da Silva


**Objetivos do projeto:** Engajar, motivar e aumentar o desempenho no ensino de Engenharia de Software em disciplinas introdutórias de programação

•	O projeto consiste em uma aplicação web, que utiliza conceitos de gamificação para aumentar o engajamento, motivação e desempenho dos alunos.

•	A aplicação possui uma breve introdução sobre a plataforma Arduino e um exercício prático, para familiarizar os alunos com a plataforma.

•	São abordados três temas neste projeto: manutenção de software, teste de software e processos de software, cada um possuindo uma introdução e um curto questionário, que deverão ser respondidos pelo menos uma vez. Após isso, em conformidade com o tema escolhido, são exibidas fases, que só poderão ser avançadas após a submissão do exercício proposto por ela.

•	Cada fase contém uma pequena descrição do projeto, objetivos, itens necessários, um esquemático para auxiliar na montagem do circuito, diagrama de caso de uso, diagrama de atividades, diagrama de máquina de estado, diagrama de classe e um sketch (código que é carregado na plataforma Arduino), que poderão sofrer alterações de acordo com o exercício proposto.
